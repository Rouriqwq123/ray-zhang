# Ray Zhang — Personal Portfolio

A space-themed interactive portfolio built with vanilla HTML, CSS, and JavaScript.

## Features

- Orbital planet navigation — four interactive planets orbit the center, each opening a content section
- Draggable planets with snap-back physics
- Animated particle starfield background
- Loading screen with fade-in transition
- Fully responsive

## Sections

- **About Me** — background and interests
- **Experience** — Fluxink internship and RS Studio
- **My Projects** — Hollow Knight (Java), Fridge AI, WinXP Portfolio, Bilibili, YouTube, RS Studio
- **Contact** — email, LinkedIn, GitHub, phone

## Tech

Pure HTML / CSS / JavaScript — no frameworks, no build step.
